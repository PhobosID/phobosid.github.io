## View the Website

You can View my Website by [Click Here](https://phobosid.github.io) or Using:

- Linux Terminal:
```terminal
xdg-open https://phobosid.github.io
```
- Windows Command Prompt:
```cmd
explorer "https://phobosid.github.io"
```
- MacOS Terminal:
```terminal
open "https://phobosid.github.io"
```
